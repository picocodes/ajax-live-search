Installation

Here's how to install Ajax Live Search:-

Download Ajax Live Search.

In your WordPress admin dashboard; click on Plugins then Add New.

Click on Upload to upload the zip file you downloaded.

Wait for WordPress to install the plugin then click on Activate Now to activate Ajax Live Search.

Ajax Live Search has now been installed. It will automagically index your blog posts in the background so you don't have to worry about anything. With time, your posts will start appearing in search results.

Settings

Ajax Live Search gives you complete control over how it works. In your WordPress admin dashboard; click on Ajax Live Search to open the plugin options.

Here are some of the things you can do:

Search term suggestions.

These options are found in the Query Completion tab and let you:-

Enable/Disable keyword suggestions.

Set the number of suggested searches to display.

Set whether to fetch the suggestions from Google, Amazon, Wikipedia, YouTube or from previous searches on your website.

Style the suggested searches.

If you are on shared hosting, avoid loading search suggestions from your local database to reduce the load on your server.

Indexing Options.

The indexing tab lets you set what type of content to include in the search index. These includes:-

Post types such as pages and blog posts.

Fields such as titles, content, comments and permalinks.

Post categories to exclude from the index.

Post tags to exclude from the index.

Post authors to exclude from the index.

You can also set the number of posts to index at a time.

Note that setting a higher number speeds up the indexing process but puts more stress on your server.

Searching Options.

These options are found in the searching tab. They let you enable or disable:-

Ajax Search. Search results are displayed without reloading the page.

Live Search. Search results are displayed while the user is still typing.

Custom Snippets. Use Google like excerpts on the search results page.

You will have to make some changes to your theme for Ajax Search and Live Search to work.

Ranking Options.

These options are found in the Ranking tab. They let you change how search results are ranked by setting weights to different:-

Post types. For example, you can give products more weight so that they appear higher in search results.

Search Field. Assign different weights to fields such as post titles and comments.

You can also set Ajax Live Search to favor new and popular content in search results.

You can also rank search results by date, title, author, popularity, post type or randomly.

Configure your theme

Due to how WordPress works, you will need to make a couple of changes to your theme's Search template before Ajax Searching or Live Searching works.

To do this:-

In your admin dashboard; click on Appearance -> Editor

Select your active theme.

Click on the file named search.php under the Theme Files menu.

Add the following code immeadiately before the line that says if ( have_posts() ) :.

 if ( function_exists('als_ajax_init') ) {
			
	als_ajax_init();
			
}

Now scroll down the page until you enter the while loop, then copy the file name that appears between get_template_part()

Next; scroll down the page until you enter the while loop, then copy the file name that appears between get_template_part()

In the official twentysixteen theme, this file is usually named template-parts/content-search

Finally, open the ajax live search settings page and click on the searching tab. Enable ajax/live search then scroll down to the template file field and insert the name that you copied in the above step.


How to get the most out of Ajax Live Search.

Enable caching to speed up searches by upto 245%.

If you are on shared hosting; avoid using Local database as the autocomplete engine to reduce server load.

Edit the autocomplete and snippet look to match your site appearance.

Giving your titles and permalinks more weight will ensure that the most relevant results appear at the top.

Query correction will only work if there is a good amount of previous searches.

Make sure that you have at least 10 posts published so that Ajax Live Search can understand the topic of your website and improve the ranking of posts.

Ajax Live Search will automatically attach itself to any search box on your WordPress website. Contact your theme's author for instructions on how to add a new search box.

How to Search.

You can search using the operators below.

+ A leading plus sign indicates that this word must be present.

- A leading minus sign indicates that this word must not be present in any of the returned results.

> Indicates that this word should be given more weight.

~ Indicates that this words weight should be negated but not removed. See 2 above.

* When you append an asterisk to a word; we match all words beginning with the value before the asterisk.

"" Returns results that contain the phrase in the parenthesis and in the same order.

author_in: Comma separated Ids of authors whose posts should be searched.

author: or @ The username of an author whose posts should be searched.

author_not_in: Comma separated Ids of authors whose posts should be excluded.

cat: or category_in Comma separated list ids of categories to search.

category: Comma, "-" or "+" separated list of category slugs to search in. Use comma to get posts from any of the categories. Use "+" to fetch posts that appear in all categories. Use "-" to exclude posts from the given category.

tagged: Comma or "+" separated list of tag slugs to search in. Use comma to get posts with any of those tags. Use "+" to fetch posts that have all those tags.

post_types: comma separated list of post types to search. The post types given should be the same as those stated in your settings page as searchable.

before: An english date format to use, e.g, last week, 2 months ago, January 1st, 2013 etc. Enclose in quotes. Checkout a list of all available formats here. This query returns posts published before the given date.

after: An english date format to use, e.g, last week, 2 months ago, January 1st, 2013 etc. Enclose in quotes. Checkout a list of all available formats here. This query returns posts published after the given date.

Let your users search directly using the above search operators or use them to create an advanced search section on your website. The limit is your imagination.

View Search Logs.

You can view a list of all previous searches by clicking on the statistics tab of the plugin settings.

This is useful when searching for new content ideas. For example, you can make a list of all popular searches returning few results and use it to write a new post.

You can also download them in csv format by clicking the blue button to the right, and then either analyse them using your favourite spreadsheet or upload them to Google Keyword planner and use it to brainstorm your next post ideas.

You can use Ajax Live Search on large websites without fear of overworking your server because it's lightweight and does not include server intensive features such as PDF indexing.
